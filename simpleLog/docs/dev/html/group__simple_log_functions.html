const unsigned int  filterLevelFile

Sets the current filter level

Parameters
filterLevelConsole- Output level to CV console. Combination of the various log levels. Messages that don't match will be ignored
filterLevelDatalog- Output level to Datalog stream. See log levels
filterLevelFile- Output level to File. See log levels. In order to output to file simpleLog_Setup should be called with proper fileName

Example and default levels: simpleLog_FilterLevel(SL_ALL, SL_NOTICE | SL_WARNING | SL_ERROR, SL_ALL)

int simpleLog_Flush ( void  )

Flushes log messages. Closes file (if open)

Should be called at the very end of the running program if simpleLog_Setup was called prior and logging to file was enabled. Also, simpleLog_Flush can be called if output to file is not needed anymore.

Returns
0 if file was closed successfully.
int simpleLog_Setup ( const char *  fileName,
const char *  timeFormat,
const char *  separator 
)

Sets the name of the log file and other parameters.

Should be called only once.
The file is opened in append mode. simpleLog_Flush must be called to close the file at the end of the program run.
Call to this function is optional same as (NULL, NULL, " ")

Parameters
fileName- Pathname of the file to write to. Pass NULL to disable output to file.
timeFormat- Optional strftime format string. Pass NULL to disable. Suggested "%Y%m%d-%H%M%S" or "%Y/%m/%d-%H:%M:%S"
separator- Character(s) used as separators between fields
Returns
0 if file was open successfully. If logging to file was not enabled function always returns 0.

Examples: (NULL, NULL, " ") print all messages no time stamp
("Msg.log", "%H:%M ", ", ")
("Msg.log", "%Y%m%d-%H%M%S ", "\t")
(NULL, "%Y%m%d-%H%M%S ", " ")

Parameters

Returns
